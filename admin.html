Import Data dari PDF

Cara Menggunakan:

  1. Buka file PDF Tarif Tol Anda.
  2. Tekan Ctrl + A (Select All) lalu Ctrl + C (Copy).
  3. Paste (Ctrl + V) teks tersebut ke dalam kotak di bawah ini.
  4. Klik tombol "Konversi ke JSON".
  5. Salin hasil kode di bawah dan masukkan ke file data/tariffs.js.
